# AtlasGo - Urban Public Transportation App

A React Native Android application for searching, planning, and purchasing public transportation routes in urban areas.

**GitHub Repository:** https://github.com/PzYusawaLux/UX-Project

## Quick Start

### Important: Use the main branch
```bash
git clone https://github.com/PzYusawaLux/UX-Project.git
cd AtlasGo
git checkout main
```

### Install the App

**Download the latest APK:** 
[AtlasGo APK](https://expo.dev/artifacts/eas/nJgzp6bfh7Vcb7cWwe93RS.apk)

### Development and Build

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build APK (requires EAS CLI and Expo account)
eas build --platform android --profile preview
```

## Features

✅ **Route Search** - Find public transportation routes in the city  
✅ **Map Navigation** - Visualize routes on an interactive map  
✅ **Ticket Purchase** - Convenient online ticketing system  
✅ **Trip Management** - View and manage purchased tickets  
✅ **Station Information** - Check departure boards and schedules
